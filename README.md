# FUTURE_ML_02
THIS IS MY TASK 02
🎫 Support Ticket Classification & Priority Prediction using Machine Learning

✨ Built as part of Machine Learning Internship – Future Interns

---

🚀 Project Overview

This project focuses on automating customer support systems by classifying support tickets into categories and assigning priority levels.
It helps organizations respond faster and improve customer experience.

---

🎯 Objectives

- Automatically classify support tickets
- Assign priority levels (High / Medium / Low)
- Reduce manual effort in support systems
- Improve response efficiency

---

🧠 Machine Learning Approach

- Converted text data into numerical features using TF-IDF Vectorization
- Trained a Logistic Regression model for classification
- Implemented rule-based logic for priority assignment

---

📊 Features

- Text preprocessing and vectorization
- Ticket classification (Billing / Technical)
- Priority prediction system
- Real-time prediction capability

---

💡 Example Output

Input: "payment not working"
Predicted Category: Billing
Priority: Low

---

📈 How It Works

1. Input ticket text
2. Convert text into numerical features (TF-IDF)
3. Model predicts category
4. Priority is assigned based on keywords
5. Output is displayed

---

🛠️ Technologies Used

- Python
- Pandas
- Scikit-learn
- TF-IDF Vectorizer
- Logistic Regression

---

📂 Project Structure

- "task02.ipynb" → Notebook with model implementation
- "model.pkl" → Saved ML model (optional)

💡 Key Learnings

- Basics of Natural Language Processing (NLP)
- Text feature extraction using TF-IDF
- Classification using Machine Learning
- Combining ML with rule-based systems
